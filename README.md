# Java Alarm Clock

A simple **command-line alarm clock** written in Java.  
It allows the user to set an alarm time, wait in real-time, and get a notification when the alarm triggers.

---

##  Features
- Set custom alarm time (hour and minute)
- Displays countdown until alarm
- Plays/prints alarm message when time is reached
- Uses multithreading (`Thread.sleep()`) or `Timer` for scheduling

---

##  Technologies Used
- **Java 17+**
- **java.util.Timer** / **Thread.sleep()**
